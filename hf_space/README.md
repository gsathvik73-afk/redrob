# ARGUS Redrob Ranker Sandbox

Upload a small candidate file and run the same offline ranking path used by the
submission repository. Supported inputs: CSV, JSON, JSONL/NDJSON, and gzipped
variants.

This hosted sandbox is for Redrob small-sample reproducibility. It does not call
hosted LLM APIs and does not need GPU access.
